# official-zh-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常以静态形式存放，不依赖特定后端服务，可直接在浏览器中打开或通过 GitHub Pages 等静态托管方式访问。

## 项目简介

这是一个通用归档仓库，主要存放各类 HTML 资源页面。页面内容涵盖工具、信息展示、实验性页面等，旨在提供一种方便、轻量的存档与分享方式。所有页面均以独立文件形式存在，便于维护与索引。

## 目录结构

```
.
├── README.md          # 本说明文件
├── index.html         # 可选：仓库首页 / 页面索引
├── pages/             # 主要 HTML 页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 公共资源（如 CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `pages/`：存放所有独立 HTML 页面文件，文件名建议使用有意义的英文或拼音，便于识别。
- `assets/`：存放页面通用的样式、脚本或图片资源，避免重复引入。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可单独访问。
- 页面之间不存在强依赖关系，但可共用 `assets/` 下的资源。
- 归档的页面可能包含实验性或临时内容，不保证长期可用或兼容所有浏览器。
- 若页面需要外部资源（如 CDN 库），建议在页面内直接引用，避免依赖仓库内资源。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来建议新增、更新或移除页面。
- 新增页面时，请确保文件命名清晰，并放置在 `pages/` 目录下。
- 如需更新公共资源，请同步检查所有引用该资源的页面是否正常。
- 本仓库不承诺定期更新，但会尽量保持页面可访问。
